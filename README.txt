FS25_USUnits - v1.0.0
=====================
Converts in-game fill amounts to US imperial units.

CONVERSIONS
-----------
  Grain crops      → Bushels (bu)      USDA standard weights per crop
  Forages / hay    → Short Tons (ton)
  Root vegetables  → Short Tons (ton)
  Fuel / liquids   → Gallons (gal)
  Operational fluids (AdBlue, oil, coolant) → Hidden

COMPATIBILITY
-------------
  Base game fill types: fully covered
  Modded fill types: auto-detected by density where possible
  Multiplayer: supported

KNOWN LIMITATIONS
-----------------
  Sell point price-per-unit display is not yet converted (shows metric).
  Contracts screen fill amounts are not yet converted.
